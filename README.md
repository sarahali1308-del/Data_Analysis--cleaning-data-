# Data_Analysis-task-2-cleaning-data-
End-to-end data project for my SAM AI Technologies internship. Includes a Python web scraper for Wikipedia company data using BeautifulSoup and an Exploratory Data Analysis (EDA) with Seaborn visualizations. Features automated data cleaning, type conversion, and a Top 10 Revenue bar chart.
Technical Stack

Python: Primary language for scraping and analysis.

BeautifulSoup: Used for parsing Wikipedia HTML.

Pandas: Applied for data cleaning, row filtering, and type conversion (e.g., transforming text to float64).

Seaborn & Matplotlib: Used to generate high-fidelity data visualizations.

📂 Project Structure

Task 1: Web Scraping: Extracted real-time company data and exported it to a structured largest_companies.csv file.

Task 2: Exploratory Data Analysis (EDA):

Cleaned the dataset by removing redundant headers and handling null values.

Performed data type transformation to enable mathematical analysis.

Visualized the Top 10 Global Companies by Revenue in a bar chart to identify market leaders.

📈 Key Insights

Successfully scraped and cleaned data for 50+ global entities.

Identified Amazon and Walmart as the leading companies by revenue in the current dataset.

Visualized the revenue gap between top-tier retail and energy sectors.
